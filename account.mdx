<Steps>
<Step title="Sign up for a Roundtable account">
    Click [here](https://app.roundtable.ai/sign-up) to create a Roundtable account and receive debugging credits.
  </Step>
<Step title="Generate API Key">
    Navigate to [manage account](https://app.roundtable.ai/account) and click on `+ Create new secret key`. Copy the API key to your clipboard and store it somewhere safe. If you lose this API key, you can always generate another one.
</Step>
</Steps>
