Roundtable Alias is an API that analyzes web and user behavior to open-end survey responses. We flag common bot patterns in responses in order to automate data cleaning and fraud detection.

## Quickstart

Easily call our API with a simple curl command

<Tip>
You can always call our most recent API endpoint at https://roundtable.ai/api/alias/latest
</Tip>

``` bash
curl --request POST \
  --url https://roundtable.ai/api/alias/v014 \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api_key>' \
  --data '{
  "participant_id": "participant_123",
  "questions": {
    "Q1": "How do you feel our software has impacted your daily workflow?"
  },
  "responses": {
    "Q1": "I really like pineapple on pizza."
  },
  "survey_id": "survey_456"
}'
```
Sign up for an [account](/account) to receive your API key.
