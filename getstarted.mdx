## Setup your account

<AccordionGroup>
  <Accordion icon="right-to-bracket" title="Sign up for a Roundtable account">
    Click [here](https://roundtable.ai/sign-up) to create a Roundtable account and receive free API credits.
  </Accordion>
  <Accordion icon="key" title="Generate API key">
    Navigate to [manage account](https://roundtable.ai/account) and click on `+ Create new secret key`. Copy the API key to your clipboard and store it somewhere safe. If you lose this API key, you can always generate another one.
  </Accordion>
</AccordionGroup>
